About

Wabisabi is a Scala Elasticsearch client that uses the REST API and has no dependency on ElasticSearch itself. It is extremely minimal, eschewing any sort of parsing, formatting or other such complication.

Why Doesn't This Library Do JSON

Because I want that to be your choice, not mine.

Use It

To use, add the following to your 

build.sbt
file. 

// Add the Dep
libraryDependencies += "wabisabi" %% "wabisabi" % "2.0.2"

// And a the resolver
resolvers += "gphat" at "https://raw.github.com/gphat/mvn-repo/master/releases/",

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
